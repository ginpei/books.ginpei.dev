Set up Prettier on ESLint

2023-07-22 (History)
  1. Install
  2. Run

Install

$ npm install --save-dev prettier eslint-config-prettier eslint-plugin-prettier

{
  "extends": ["next/core-web-vitals", "prettier"],
  "plugins": ["prettier"],
  "rules": {
    "prettier/prettier": "error"
  }
}

Run

$ npx eslint --fix .
